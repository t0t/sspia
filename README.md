# SSPIA
**Static Single Page Invoice Application** that uses Web Standards.

![Screenshot](./screenshot.png)

## Raw Standards Only

  - HTML
  - JS/ES6
    - Modules
    - Classes
    - MVC
  - CSS
  - SASS
    - Components
    - BEM
  - [The DOM](https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model)
  - JSON as "DB"

## Development workflow

 * Atom
 * Git
 * Gulp & NPM:
  - Deploy to gh-pages
  - Watch
  - Babel
  - BrowserSync
  - Autoprefixer
  - Concat
  - Minificado
